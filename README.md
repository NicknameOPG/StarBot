# Comet
Comet is Starbot's new name, deal with it
# depenedcies
- axios
- discord.js
- colors
- mongoose
- canvacord
# usage
to use this bot:
make a file called `config.json` and put this inside it
```js
{
   "token": "bot token",
   "clientId": "bot id",
   "prodiac": "prodia api key",
   "mongo": "your mongodb url"
}
```
- in console 'npm run start'
# contribute
- fork the bot

- wait for it to get accepted/denied
